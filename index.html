Welcome to PXF extensions for HAWQ!

As a means to share knowledge, code, and save time (I hear it = money), the following pages are a collective effort of the field to build extensions for HAWQ. Feel free to add and collaborate.

PXF prototype extensions are hosting in the pxf-field GitHub repository for Pivotal Field Engineering. You can clone this directly using the below command if you have a Pivotal Field Engineering GitHub account. If you don't, go to https://github.com/Pivotal-Field-Engineering and create one.

git clone https://github.com/Pivotal-Field-Engineering/pxf-field.git
